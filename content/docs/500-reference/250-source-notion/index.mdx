This module provides the methods for working with [Notion](https://notion.so) as your content source.

Typical usage involves two primary functions:

- [`makeSource`](/docs/reference/source-notion/make-source) controls the entire schema definition for your application.
- [`defineDatabase`](/docs/reference/source-notion/define-database) defines how your database should be used to generate your document types.
